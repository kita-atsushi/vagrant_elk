## Usage

以下を`vagrant up`で作成したVM内で実行します(1時間程度かかりますので注意ください)
```
sudo su -
bash machine-learning-nyc-taxi-demo.sh

```

elastic社のリンクに合わせた機械学習機能を利用できる状態になります。  
Kibanaの画面を使用して機械学習機能を使用ください。
